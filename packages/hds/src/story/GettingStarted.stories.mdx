# Highoutput Venture Design System

**CURRENT VERSION:** 0.4.8

**@highoutput/hds** is a ui component library that is built around [chakra ui](https://chakra-ui.com/), and aims to be:

- a reusable block of components
- very customizable with ease of styling
- easy to implement

## Getting Started

To install package, use:

```bash
npm i @highoutput/hds
```

### Usage

```typescript
import * as React from 'react';
import * as ReactDOM from 'react-dom';
import { ThemeProvider } from '@highoutput/hds';

const App = () => {
  return (<ThemeProvider><div>{...components here}</div></ThemeProvider>);
};

ReactDOM.render(<App />, document.getElementById('root'));
```
